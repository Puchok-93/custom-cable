BNC

Коаксиальный коннектор с быстрым соединением/разъединением, которое выполняется вращением на четверть оборота сцепляющей гайки.

HDMI

Интерфейс для мультимедиа, позволяющий передавать цифровые видеоданные и цифровые аудиосигналы с защитой от копирования.

IEC-C7

Разъем для силового кабеля, часто используют для небольших кассетных магнитофонов, радиоприёмников с батарейно-сетевым питанием.

IEC-C13

Трёхпроводный разъем для силового кабеля, используется для питания многих видов электроприборов.

IEC-C15

Трёхпроводный разъем для силового кабеля, используется для питания многих видов электроприборов.

Lighting

Разъём используемый в портативной технике Apple. Передаёт только цифровые сигналы, вставляется в устройство обеими сторонами.

Mini DisplayPort

Миниатюрная версия аудиовизуального цифрового интерфейса DisplayPort. Анонсирован Apple в октябре 2008 года.

ММСХ

Разъем для передачи аудиосигнала для вставных наушников. Обычно на другой стороне кабеля стерео штекер 3,5 мм.

RCA

Разъем для передачи аудиосигнала. Используется для соединения аудио-компонентов, таких как усилитель, ЦАП, проигрыватель CD и других.

RJ-45

Унифицированный разъём, используемый в телекоммуникациях / для создания ЛВС. Имеет 8 контактов и фиксатор.

Spade

Spade или «лопатка». Выполнен из бескислородной меди с родиевым покрытием. Разновидность разъема для подключения к акустическим системам.

Thunderbolt

Разъем, разработанный Intel в сотрудничестве с Apple. Служит для подключения различных периферийных устройств к компьютеру.

Toslink

Стандарт соединения с помощью оптоволокна (световодов), также широко известный как «оптический аудио кабель».

TRS, Jack, 6.3

Разъём для передачи аналогового аудиосигнала. Применяется преимущественно для музыкальных инструментов, микрофонов и клавишных.

USB Type A

Спецификация USB для универсального подключения к компьютеру, гаджетам и аудио-устройствам.

USB Type B

Спецификация USB для универсального подключения к компьютеру, гаджетам и аудио-устройствам.

USB Type-C

Спецификация USB для универсального двустороннего подключения к компьютеру, гаджетам и аудио-устройствам.

Micro-USB

Спецификация USB для универсального подключения к компьютеру, гаджетам и аудио-устройствам.

Mini-USB

Спецификация USB для универсального подключения к компьютеру, гаджетам и аудио-устройствам.

VGA

Аналоговый разъём для подключения мониторов по стандарту видеоинтерфейса VGA (Video Graphics Array).

XLR

Балансный разъем, используется в профессиональном и Hi-End оборудовании для повышения устойчивости к помехам и повышения качества звучания.
